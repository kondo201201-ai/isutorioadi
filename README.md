# お互いを知ろう お題入力アプリ

9月の集会で使う「お互いを知ろう」椅子取りゲーム版のお題入力アプリです。

## 開き方

`index.html` をダブルクリックすると起動します。

## できること

- 1人3つのお題をまとめて登録
- 登録したお題の検索
- 1件ずつ削除
- Googleスプレッドシートに貼り付けやすい1列テキストをコピー
- CSV保存
- 入力内容をブラウザに自動保存

## 使い方

1. 名前またはニックネームを入れます。
2. お題を3つまで入力します。
3. 「登録する」を押します。
4. 右下の貼り付け用データをコピーして、GoogleスプレッドシートのA1に貼り付けます。
